How popular is it on Twitter?

Find the popularity of a phrase on Twitter, live!

Type in the phrase and hit the button

You will have to wait for a few minutes while we gather data and compute the score

Meanwhile, you can go read about how the score in computed here

The popularity of '{{ phrase }}' is {{ x }}
{% endif %}

The formula is still a bit wibbly wobbly and it will not give you an absolute determinstic score.
However, a few other phrases that you think are way more (or less) popular will help you get a qualitative (or relativistic) sense.

The default timeout is 5 minutes. So you will have to wait for 5 minutes to get a score.
If you are not getting any result after the time limit, it most probably means that the phrase was not tweeted during the interval.

If you think you broke the app (or found a bug), report it via GitHub or mail.
